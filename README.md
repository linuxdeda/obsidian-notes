# Obsidian Notes

Ovaj repozitorijum sadrži moj Obsidian vault – beleške, ideje, projekti i dokumentacija na kojima radim svakodnevno.

📘 Šta je uključeno

- Beleške organizovane po folderima
- Interne poveznice i graf prikazi
- Template-i i korisni snippeti
- Dodatni fajlovi povezani sa raznim projektima

## 🔧 Korisni alati
Koristim dodatak **Obsidian Git** za automatsko čuvanje i sinhronizaciju beleški sa GitHub repozitorijumom.

## ⬆️ Git workflow
Obsidian Git plugin automatski:
- pravi commit kad se beleške promene
- push-uje na GitHub
- omogućava pregled istorije i revert

## 📝 Napomena
Vault može sadržati privatne ili radne beleške, pa je struktura prilagođena mom načinu rada. Ako preuzimaš ili koristiš nešto, proveri licencu i privatne informacije.
